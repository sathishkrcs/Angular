Defining Colors in CSS:
///////////////////////
In the past,hexadecial notation or color keywords have been used to define color


CSS 2.1 allowed color to be defined as RGB Values


CSS3 adds the SVG 1.0 color keywords to the
specification

for more on availble keywords visit the color module
http://www.w3.org/TR/css3-color/#svg-color


New Color FORMATS IN CSS 3

RGB (in CSS2)
red,green,blue

RGBA
red,green,blue,alpha

HSL
hue,saturation,lightness

HSLA
hue,saturation,lightness,alpha

CMYK
cyan,magenta,yellow,black

RGB Syntax:
RGB is an additive color model. Red,green and blue are mixed together to
form colors.
As an additive method,100% of all three colors would
produce white.

In CSS,RGB values can be written as three comma-separated
values ranging from 0-255, or percentages
ranging from 0- 100%
